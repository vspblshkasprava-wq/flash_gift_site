# Реальные звуки

Сейчас игра сама генерирует мягкие звуки через браузерный Web Audio API. Старые варианты звуков тоже оставлены в меню.

Если позже появятся готовые файлы, положите их в эту папку с такими именами:

- `pop-soft.mp3` или `pop-soft.ogg`/`pop-soft.wav` - звук лопания пузыря.
- `charge-soft.mp3` или `charge-soft.ogg`/`charge-soft.wav` - длинный звук накопления левого суперудара.

В текущей версии код уже пробует варианты `pop-soft.mp3`, `pop-soft.ogg`, `pop-soft.wav`, `charge-soft.mp3`, `charge-soft.ogg` и `charge-soft.wav` через меню звуков. Если файла нет или браузер не смог его проиграть, игра автоматически возвращается к встроенному мягкому звуку.

Для iPhone/iPad лучше использовать короткие MP3 или AAC/M4A. Для ПК обычно подходят MP3, OGG и WAV.
