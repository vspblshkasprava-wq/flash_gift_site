# Игры для малыша

Статический браузерный проект без установки, сборки, `npm`, CDN и серверной логики. Главный вход: `index.html`.

Проект можно загрузить на обычный хостинг как набор HTML/CSS/JS-файлов. Планируемый адрес: `https://flash.gift/kids/`.

## Что внутри

- `index.html` - главное меню.
- `styles.css` - стили главного меню.
- `keyboard-game/` - отдельная игра для клавиатуры.
- `bubble-game/` - игра с мыльными пузырями, мышкой, тач-управлением и уровнями.
- `assets/icons/` - иконки для браузера, iPhone/iPad и PWA.
- `assets/sounds/` - место для будущих реальных звуков.
- `manifest.webmanifest` - manifest для добавления на экран телефона/планшета.

## Как открыть локально

Можно открыть `index.html` напрямую в браузере.

Для проверки через локальный сервер откройте PowerShell или терминал в папке проекта и выполните:

```powershell
python -m http.server 8765
```

После этого откройте:

```text
http://127.0.0.1:8765/
```

Если команда `python` не работает, установите Python или откройте PowerShell в папке проекта и используйте установленный Python из вашей системы. Не нужно менять файлы проекта под конкретный путь Python.

## Игра с пузырями

Сейчас есть уровни:

1. `Свободные пузыри` - мягкий бесконечный режим без таймера.
2. `Звездная палочка` - яркий режим со счетом, 90 секундами и целью 10000 очков.
3. `Палочки-надуватели` - пузыри появляются около палочек по краям.
4. `Цветные пузыри` - бонусные пузыри: золотые дают очки, голубые замедляют, звездные выпускают звездочки.
5. `Веселый хаос` - больше пузырей и эффектов, цель 15000 очков.

Раунд заканчивается по таймеру или досрочно по цели очков. После конца уровня появляется окно: повторить, следующий уровень или меню.

## Управление мышкой

- Левая кнопка - плавное увеличение палочки до 500%, удержание на максимуме около 5 секунд, затем новый цикл.
- Правая кнопка - радужная волна.
- Колесо мыши - вихрь вокруг палочки.
- Нажатие колеса - взрыв пузырей.
- Боковая кнопка назад - заморозка.
- Боковая кнопка вперед - звездный дождь.

На телефоне/планшете движение пальцем лопает пузыри. В меню можно включить экранные кнопки `Супер` и `Волна`.

## Звуки

В меню игры с пузырями есть выбор звука лопания и звука накопления левого суперудара. Старые варианты сохранены, новые мягкие варианты добавлены. Есть ползунки общей громкости и громкости суперударов.

Для реальных файлов положите в `assets/sounds/` один или несколько файлов:

- `pop-soft.mp3`, `pop-soft.ogg` или `pop-soft.wav`
- `charge-soft.mp3`, `charge-soft.ogg` или `charge-soft.wav`

После этого выберите в меню звука вариант `Файл: pop-soft` или `Файл: charge-soft`. Если файл не найден или браузер не может его проиграть, игра сама вернется к встроенному мягкому звуку.

## Сохранение результата

Игра сохраняет настройки и прогресс в `localStorage` браузера:

- выбранный уровень и предмет;
- включение клавиатуры поверх пузырей;
- цвет обводки букв;
- выбранные звуки;
- громкость;
- включение мобильных супер-кнопок;
- запуски уровней, лучший счет и факт завершения.

Сброс есть в меню игры с пузырями в блоке `Прогресс`.

## iPhone и iPad

Для добавления на экран `Домой` обычно нужна публикация по HTTPS. После публикации откройте адрес в Safari и выберите `Поделиться` -> `На экран "Домой"`.

Браузер на iPhone/iPad не дает надежно заблокировать системные жесты и часть системных действий, но обычные действия внутри страницы перехватываются игрой, пока она открыта и активна.

## Выход

В обеих играх:

- `Ctrl + X` возвращает в меню.
- Мышкой можно выйти, удерживая правый верхний угол около одной секунды.

Системные сочетания Windows вроде `Win`, `Alt + Tab` и `Ctrl + Alt + Del` обычная веб-страница надежно заблокировать не может.

6. Проверить старые звуки.

8. Проверить кнопку проверки звука.
9. Проверить громкость.
10. Проверить левый суперудар.
11. Проверить сохранение лучшего счета после перезагрузки.
12. Проверить сброс прогресса.
13. Проверить `keyboard-game` на ПК.
14. Проверить `Ctrl + X`.
15. Проверить выход удержанием правого верхнего угла.

17. Проверить на iPad Safari.
